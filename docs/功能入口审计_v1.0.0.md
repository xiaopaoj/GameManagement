## 入口行为约束

- 已实现功能执行实际操作并提供成功或错误反馈。
- 当前审计范围内不再保留仅显示开发状态的占位按钮。
- 不允许保留无响应按钮。
- 文件扫描期间必须显示百分比和当前路径，禁用主界面其他入口并阻止关闭主窗口，扫描结束后自动恢复。
- 扫描候选的只读属性必须使用单向绑定，扫描结果渲染不得因 WPF 尝试回写只读属性而退出应用。
- 阻塞式任务和临时禁用窗口的异步任务必须在结果提示前恢复调用窗口，并在进度窗口关闭后保持调用窗口位于前台；当前回归测试总数为 72 项。
- 后续版本新增或调整功能时，应同步更新本审计表并补充相应测试。
- “操作历史/删除历史”已实现回收站、永久删除和数据库记录删除结果查看。
