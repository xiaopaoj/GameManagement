## 入口行为约束

- 已实现功能执行实际操作并提供成功或错误反馈。
- 后端尚未实现的入口打开功能状态页，明确说明当前不会进行复制、解压、覆盖或删除。
- 不允许保留无响应按钮。
- 文件扫描期间必须显示百分比和当前路径，禁用主界面其他入口并阻止关闭主窗口，扫描结束后自动恢复。
- 后续实现功能后，应将本审计表的状态同步修改为“已实现”，并补充相应测试。
