# 功能入口审计

版本：1.0.0  
审计日期：2026-07-19

最终审计结果：需求定义的界面入口均已连接实际功能，不再保留开发状态占位按钮或无响应按钮；60 项 Release 自动化测试及图形界面、扫描结果渲染、计划备份模式冒烟验收全部通过。

安全密码模式已完成数据库、游戏图标和运行日志强制加密，以及密码包装和锁定生命周期：主数据库使用 AES-256-GCM，普通模式主密钥由当前 Windows 用户 DPAPI 保护；开启安全模式后使用 PBKDF2-SHA256 派生密钥包装主密钥，启动前先显示不加载真实数据的最小包装外壳，支持设置、修改、关闭密码、立即锁定、可配置空闲自动锁定和持久化错误等待，未解锁时禁止计划备份。旧明文数据库、图标和日志迁移均先完成加密回读校验，删除仍需人工确认；加密日志通过软件内置查看器读取，锁定时清除已解密图标的内存缓存。包装业务功能暂不设计。

## 入口行为约束

- 已实现功能执行实际操作并提供成功或错误反馈。
- 当前审计范围内不再保留仅显示开发状态的占位按钮。
- 不允许保留无响应按钮。
- 文件扫描期间必须显示百分比和当前路径，禁用主界面其他入口并阻止关闭主窗口，扫描结束后自动恢复。
- 扫描候选的只读属性必须使用单向绑定，扫描结果渲染不得因 WPF 尝试回写只读属性而退出应用。
- 阻塞式任务和临时禁用窗口的异步任务必须在结果提示前恢复调用窗口，并在进度窗口关闭后保持调用窗口位于前台；可后台运行任务的公共进度窗口支持最小化、后台运行和任务栏恢复。
- 后续版本新增或调整功能时，应同步更新本审计表并补充相应测试。
- “操作历史/删除历史”已实现回收站、永久删除和数据库记录删除结果查看。
